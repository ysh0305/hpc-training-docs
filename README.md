# HPC Training Website

This repository contains the source code for the **HPC Training Website**.

The site is split into:
- a **main website** built with **Next.js**
- a **documentation & blog site** built with **Docusaurus**

---

## Repository Structure

``
repo/
├─ apps/
│  ├─ web/        # Next.js main website
│  └─ docs/       # Docusaurus documentation
└─ .github/
   └─ workflows/  # CI/CD (build & GitHub Pages deployment)
``
